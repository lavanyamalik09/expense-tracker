# Expense Tracker

## Description
This is a simple expense tracker built using Python and SQLite.
It helps users add, view, and analyze their daily expenses.

## Features
- Add expenses
- View all expenses
- Show total expense
- Filter by category
- Monthly report
- Export to CSV

## Technologies Used
- Python
- SQLite

## How to Run
1. Open terminal
2. Go to project folder
3. Run:
   py main.py
